# フィットネスデータSQL分析
このプロジェクトは、Python（pandas）とSQLを用いてフィットネステストデータを分析します。

## 機能
- フィットネステストデータセットの読み込み
- データクリーニング
- 基本的な統計分析
- データ集計のためのSQLクエリ

## 使用ツール
- Python
- pandas
- SQL

## 目的
このプロジェクトは、データ分析とデータ処理の学習の一環として作成されました。
